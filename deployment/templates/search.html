Explore predictions


This function can be used to explore past prediction accuracy. For example, select 45 for "past days" and 45 for "prediction window" to see predictions from 45 days ago, till today.


This allows you to explore how accurate the model is when considering past data.


Please note that the model is trained with 45 days prior data to predict the next 1 days price. Therefore the model validity decreases as the window increases.
